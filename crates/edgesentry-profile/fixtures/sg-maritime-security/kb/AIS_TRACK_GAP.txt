AIS Track Gap — SOLAS Continuous AIS Requirement

Under SOLAS Chapter V Regulation 19, all vessels of 300 GT and above engaged on international
voyages must operate their Automatic Identification System (AIS) continuously while underway.
Within Singapore port limits, any AIS gap exceeding 8 minutes (480 seconds) must be reported.

Continuous operation requirements:
- AIS Class A transponder must be active from departure to arrival
- AIS must not be switched off except in exceptional security circumstances
- Any AIS gap within port limits triggers immediate investigation

Reporting procedure for AIS gaps:
1. Notify MPA Vessel Traffic Information System (VTIS) immediately via VHF Ch 22
2. Submit AIS Gap Incident Report within 4 hours of gap occurrence
3. Provide reason for gap (equipment failure, interference, maintenance)
4. Submit AIS equipment maintenance record if technical fault claimed

Consequences of unexplained AIS gaps:
- Port State Control inspection on arrival
- Detention pending investigation
- Referral to flag state administration
- Potential ISPS Code compliance investigation

Reference: SOLAS Chapter V Regulation 19 (AIS)
