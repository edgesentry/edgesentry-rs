Crew Documentation — MLC 2006 Requirements

The Maritime Labour Convention 2006 (MLC 2006) requires that all seafarers on board hold
valid documentation and that crew records are complete and accurate for every crew member listed.

Required documentation per crew member:
- Valid Seafarer's Identity Document (SID) or Seaman's Passbook
- Certificate of Competency (CoC) appropriate to rank
- Medical fitness certificate (ENG1 or equivalent)
- Flag state endorsement where required

Vessel obligations:
- Maritime Labour Certificate (MLC) must be current and on board
- Declaration of Maritime Labour Compliance (DMLC) Parts I and II required
- Crew Agreement must be signed by all seafarers

Crew count discrepancies:
- Actual crew on board must match the crew count declared in pre-arrival notification
- Any variation requires immediate notification to MPA Port Formalities
- Missing crew documentation triggers Port State Control inspection

Reference: Maritime Labour Convention 2006, Standard A1.2
